# Messages to send — Two phone numbers explanation

## Can the same number be used for both purposes?

**No.** Once a number is registered with WhatsApp Business API, it is fully consumed by WhatsApp and can no longer receive regular SMS text messages. So +971 52 558 6541 (the API number) physically cannot receive the Facebook security verification code.

However, Badar CAN change his Facebook developer account security phone to any other number he has access to — it just needs to be able to receive a standard SMS.
